# AR Foundation package samples

Learn about AR Foundation package samples.

The AR Foundation package provides samples which you can import from the Package Manager to use in your project.

## Import package samples

Import AR Foundation package samples from the [Package Manager](xref:um-upm-ui):

1. In the **Package Manager** window (**Window** > **Package Management** > **Package Manager**), select **AR Foundation** from the **In Project** tab.
1. Navigate to the **Samples** tab.
1. **Import** each of the samples you want to use.

Access imported samples from `Assets/Samples/AR Foundation/`<code class="long_version">X.Y.Z</code>.

## Package samples reference

The following sections outline the package samples AR Foundation provides.

### Debug Visualizers

Runtime debug visualizers for AR Foundation trackables. Assign the visualizer to the **Prefab** slot of the corresponding trackable manager to visualize trackable behavior and assist with runtime debugging.

Refer to the following pages for more information about how to use these samples:
