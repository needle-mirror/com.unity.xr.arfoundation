# Samples

Learn about the samples Unity provides to help you develop with AR Foundation.

Unity provides two types of samples you can access and use to implement AR Foundation features:

| Topic          | Description                              |
| :------------- | :--------------------------------------- |
| [AR Foundation Samples](xref:arfoundation-samples-project) | Unity's official sample app for AR Foundation demonstrates how to set up and use AR Foundation features. |
| [AR Foundation package samples](xref:arfoundation-package-samples) | Individual samples provided with the AR Foundation package. Import sample assets to help you develop with specific AR Foundation features. |
